# ChrisHit3D
Only for fun!

Open-source under MIT license.

## Roadmap:
- Android/IOS support
- Cross platform I/O
- Data driven system
- Skeletal Animation
- Default material system
- Forward and Deferred Rendering(ES 3)

"ChrisHit", in memorian Christopher Hitchens 1949-2011.
